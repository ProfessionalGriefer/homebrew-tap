# ProfessionalGriefer's Tap

## How do I install these formulae?

`brew install ProfessionalGriefer/tap/<formula>`

Or `brew tap ProfessionalGriefer/tap` and then `brew install <formula>`.

Or, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "ProfessionalGriefer/tap"
brew "<formula>"
```

## Installable formulae

| Formula                                                       | Description                                                    |
| ------------------------------------------------------------- | -------------------------------------------------------------- |
| [`anki-tui`](https://github.com/professionalgriefer/anki-tui) | Keyboard-driven terminal reviewer for Anki (via AnkiConnect)   |
| `quickposition`                                               | Quickly position an iPad to the left or right of a Mac display |
